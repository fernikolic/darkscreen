# Error Message Patterns

Error messages collected from 105 apps.

### Emerging

- Page not found — 11% (12 apps)
- 404 — 7% (7 apps)
- 404 Not Found — 7% (7 apps)
- This page could not be found. — 5% (5 apps)
- 404: NOT_FOUND — 3% (3 apps)

### Outliers

- Page Not Found. (2 apps)
- 404 Page Not Found (2 apps)
- The page you are looking for doesn't exist or has been moved. (2 apps)
- Sorry, you have been blocked (2 apps)
- File not found (2 apps)
- The site configured at this address does not contain the requested file. (2 apps)
- The page you're looking for could not be found. (1 apps)
- We couldn't find the page you were looking for (1 apps)
- 403 Error (1 apps)
- Access to blockchair.com is forbidden. (1 apps)
