# Component Catalog

UI components identified across 105 crypto apps.

## Dominant Components (70%+ of screens)

### hero-section
- **Adoption**: 79% (83 apps)
- **Notable**: Generous vertical whitespace emphasizing the portfolio value proposition; Mission-driven copy with contrasting button treatments; Uses inline hyperlinks ('Learn more about the risks') for contextual help
- **Examples**: 1inch (center), 1inch (center), aave (Top of page, below nav), across (left-center), aerodrome (left 50% of screen)

### navigation-bar
- **Adoption**: 75% (79 apps)
- **Notable**: Connect wallet button styled as primary CTA with black background; Consistent across all pages; Swap tab appears highlighted/active
- **Examples**: 1inch (top), 1inch (top), 1inch (top), 1inch (top), aave (below banner)
