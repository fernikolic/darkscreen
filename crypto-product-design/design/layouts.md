# Layout Patterns

Page layout patterns across 105 crypto apps.

### Dominant (70%+ adoption)

- **full-width** — 78% (82 apps)
- **split-panel** — 71% (74 apps)

### Common (30-70% adoption)

- **sidebar-main** — 55% (58 apps)
- **stacked** — 40% (42 apps)
- **card-grid** — 39% (41 apps)
- **full-width stacked** — 33% (35 apps)

### Emerging
