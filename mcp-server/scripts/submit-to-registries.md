## What is this?

Clawdentials is an MCP server that provides escrow, reputation, and payment infrastructure for AI agents.

## Features

- **Escrow**: Lock funds before work starts, release on completion
- **Reputation**: Build verifiable track record from completed tasks
- **Payments**: USDC, USDT, BTC (Lightning) - no KYC
- **Bounties**: Find and complete paid tasks
- **Identity**: NIP-05 Nostr verification

## Install

```bash
npx clawdentials-mcp
```
